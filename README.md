# 숫자 랜덤 게임
1. 1~100의 범위 내에서 숫자를 맞추는 게임<br>
2. 숫자는 랜덤으로 출력되고, 범위 밖의 숫자를 입력하면 작동하지 않는다.<br>
3. 내가 입력한 숫자가 정답보다 큰지, 작은지 컨테이너에 출력된다.<br>
4. 승패 결과에 따라 나오는 이미지가 다르다.
